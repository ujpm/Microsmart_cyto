# MicroSmart PF Frontend

This folder contains the React + Vite dashboard for MicroSmart PF.

## Quick Start

```bash
npm install
npm run dev
```

Visit [http://localhost:5173](http://localhost:5173) after starting the backend server.

## Tech Stack
- React
- Vite
- Tailwind CSS

## Development
- Edit UI components in `src/components/`
- Custom hooks in `src/hooks/`
- API calls use `/analyze` endpoint from the backend

For full project documentation, see the main `README.md` in the root folder.
